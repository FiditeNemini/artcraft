local development setup
=======================

Ubuntu 22.04
------------


### Install Javascript things

```bash
sudo apt install npm
sudo npm install --global yarn
```

### Run the website(s)

Install dependencies

```bash
yarn install
```

Start development server for FakeYou

```bash
yarn start-fakeyou
```
