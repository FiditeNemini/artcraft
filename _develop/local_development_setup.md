local development setup
=======================

Ubuntu 22.04
------------

### Install NPM and Yarn

```bash
sudo apt install npm
sudo npm install --global yarn
```

### Develop and run locally

Install dependencies

```bash
yarn install
```

Start development server for FakeYou

```bash
yarn start-fakeyou
```
