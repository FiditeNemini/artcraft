aichatbot-sidecar
=================

Sidecar companion to AiChatBot. 

### Installation on Linux

The egui/eframe libraries require the installation of a metric ton of dependencies, so have at it:

```
sudo apt-get install \
  cmake \
  libfontconfig1-dev \
  libspeechd-dev \
  libssl-dev \
  libxcb-render0-dev \
  libxcb-shape0-dev \
  libxcb-xfixes0-dev \
  libxkbcommon-dev
```
